Общая архитектура
Программа состоит из одного класса CurrencyConverter, который управляет всем приложением. Используется библиотека tkinter для GUI, requests для HTTP-запросов к API, json для работы с файлами.

Основные компоненты
GUI интерфейс

Выбор валют — выпадающие списки (ttk.Combobox) с популярными валютами

Поле ввода суммы — текстовое поле (ttk.Entry)

Кнопка конвертации — при нажатии вызывает метод convert()

Таблица истории — виджет ttk.Treeview с колонками (дата, из какой валюты, в какую, сумма, результат, курс)

Работа с API

Используется бесплатный сервис exchangerate-api.com

URL: https://v6.exchangerate-api.com/v6/{API_KEY}/latest/{валюта}

Пример запроса: GET /latest/USD → получаем курсы всех валют относительно USD

Метод convert() отправляет запрос и извлекает нужный курс из поля conversion_rates

Сохранение истории в JSON

Файл conversion_history.json

Метод save_history_to_file() сохраняет список словарей в JSON

Метод load_history() загружает историю при запуске

Каждая запись содержит: дату, валюты, сумму, результат, курс

Автосохранение происходит после каждой конвертации

Проверка корректности ввода

Сумма не должна быть пустой

Преобразование в число с проверкой через try/except

Проверка на положительность: amount <= 0

Как получить API-ключ (exchangerate-api.com)
Перейдите на exchangerate-api.com

Нажмите "Get Free API Key"

Введите email и подтвердите

Бесплатный ключ придет на почту (1000 запросов/месяц достаточно)

Вставьте ключ в код: self.api_key = "ваш_ключ"

Возможные тесты
Конвертация 100 USD → EUR (должно получиться ~92 EUR)

Ввод отрицательного числа (-50) → ошибка "Сумма должна быть положительной"

Ввод букв ("abc") → ошибка "Введите корректное число"

Очистка истории → таблица становится пустой

Конвертация без интернета → ошибка сети
